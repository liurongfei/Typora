# 索引

## 索引是什么

* 索引帮助Mysql高效获取数据的数据结构
* 索引存储再文件系统中
* 索引的文件存储形式与存储引擎有关
* 索引文件的结构：
  1. hash
  2. 二叉树
  3. B树
  4. B+树



## 索引的类别

1. NORMAL(普通索引):

> mysq中的基本索引类型。在建表时会自动为主键索引创建一个NORMAL索引，

1. UNIQUE(唯一索引)

2. FULLTEXT(全文索引)

3. SPATIAL(空间索引)

## 回表

使用普通索引时查询到主键后，需要回表到聚簇索引查找行数据

## 索引覆盖

索引覆盖是一种避免回表查询的优化策略

即将要查询的数据作为索引，这样查询时直接返回索引得到数据，而不需要再去聚簇索引定位行记录
